# Future Features & Vision

This page tracks long-term ideas for KOVIL MAP.

> Note: these items are conceptual and do not have delivery dates. For current work, see the backlog.

## Long-Term Vision

The end goal is a collaborative WiFi auditing platform where collection, analysis, and cracking are tightly integrated.

## Scaling

Focus areas:

- distributed cracking across multiple machines
- real-time collaboration
- encrypted cloud sync between devices

## Intelligence

Potential future work:

- AI-assisted target selection
- rule-driven workflows and automation

## Platform Expansion

- a mobile companion app
- a plugin marketplace

## How to Influence the Roadmap

1. open an issue with your use case
2. reference related backlog or roadmap items when possible
3. contribute code or documentation when possible
