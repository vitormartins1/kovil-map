# Roadmap

This section tracks planned directions for KOVIL MAP. It is intentionally forward-looking and may include ideas that are not committed yet.

## Timeline Snapshot

| Period | Focus | Examples |
|---|---|---|
| Q2 2026 | Workspace polish | WarDrive and Analytics UX refinement |
| Q3 2026 | Intelligence | targeting automation, richer analysis |
| Q4 2026 | Scale | collaboration, auditability, packaging improvements |

## How to Influence the Roadmap

1. review the public backlog first
2. propose new ideas with clear operational context
3. document requirements before implementation
4. contribute code or docs through PRs when feasible
